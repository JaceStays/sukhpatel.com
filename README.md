# sukhpatel.com

Starter website for Sukh Patel, Real Estate Analyst in Vancouver.

## Structure

- `index.html`: Main landing page
- `assets/css/styles.css`: Site styling
- `assets/js/main.js`: Minor page behavior
- `assets/images/`: Visual assets (hero background, favicon)

## Deploy

This is a static site and can be deployed directly to Netlify.
